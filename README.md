# ESP32 步进电机控制器（GPIO 控制 + 串口终端指令）
基于 ESP32 开发的步进电机控制项目，使用 **GPIO 硬件控制** 电机驱动，通过 **ESP-IDF 串口终端** 发送指令实现交互控制。

## 功能说明
- 使用 GPIO 输出控制步进电机驱动板（DIR/STEP/EN/MS1/MS2）
- STEP 脉冲采用 ESP32 LEDC PWM 生成
- 支持通过 ESP-IDF 串口终端输入指令控制电机
- 支持启停、正反转、调速、细分设置
- 支持自动循环正反转模式
- 支持持续旋转模式

## 硬件连接（GPIO 分配）
| 驱动引脚 | ESP32 GPIO |
|---------|------------|
| DIR     | GPIO4      |
| STEP    | GPIO5      |
| MS2     | GPIO6      |
| MS1     | GPIO7      |
| EN      | GPIO15     |

## 支持的细分模式
- 1/8 细分
- 1/16 细分
- 1/32 细分（默认）
- 1/64 细分

## 开发环境
- ESP-IDF (v5.5.4)
- C 语言开发
